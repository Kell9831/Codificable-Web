Tutorial: Cómo Crear un Temporizador en JavaScript, HTML y CSS

En este tutorial, aprenderemos a crear un temporizador utilizando JavaScript, HTML y CSS. Un temporizador es una herramienta útil para contar hacia atrás desde un tiempo especificado. Puede ser útil en una variedad de aplicaciones, como aplicaciones de cocina, ejercicios y más.

Al final de este tutorial aprenderás a crear un temporizador como este:

Con la capacidad de poder cambiar el diseño a tu manera y completamente funcional para que lo puedas usar en tus aplicaciones web :D.

HTML: Creando la Estructura

Comencemos creando la estructura básica en HTML para nuestro temporizador. Aquí está el código HTML: 

<div>
  <h2 id="timer__title">Temporizador</h2>
  <div>
    <input type="number" id="inputTime" placeholder="00"/>
    <span id="segundos">segundos</span>
    <button id="empezar">Empezar</button>
  </div>
  <div id="display">--</div>
  <div>
    <button id="continuarPausar">Pausar</button>
    <button id="detener">Detener</button>
  </div>
</div>

En este código, hemos creado un contenedor principal con un título "Temporizador", un campo de entrada para el tiempo en segundos, un botón "Empezar" y dos botones "Pausar" y "Detener" con su respectuvo id para ser usados más tarde. El elemento <div id="display"> mostrará el tiempo restante.

Si bien los botones de acción como "Empezar", "Pausar" y "Detener" o equivalentes son necesarios para interactuar correctamente con el temporizador, la división entre estos usando varios elementos <div> puede modificarse según el diseño deseado.

JavaScript: Funcionalidad del Temporizador

A continuación explicaremos paso a paso el cómo implementar nuestro temporizador básico.

  1. Declaración de Variables Iniciales:

    2. let timer = 0;
let isRunning = false;
let remainingSeconds = 0;
    • timer: Esta variable se utilizará para almacenar el identificador del intervalo que ejecuta la función updateTimer para contar el tiempo.
    • isRunning: Indica si el temporizador está en funcionamiento o no.
    • remainingSeconds: Almacena la cantidad de segundos restantes en el temporizador.

  2. Selección de Elementos HTML:

    3. let empezarButton = document.getElementById("empezar");
let continuarPausarButton = document.getElementById("continuarPausar");
let detenerButton = document.getElementById("detener");
let timerInput = document.getElementById("inputTime");

    Estas variables se utilizan para seleccionar los botones y el elemento de entrada de tiempo en el documento HTML utilizando su ID.

  3. Estado Inicial de los Botones:

    4. continuarPausarButton.disabled = true;
detenerButton.disabled = true;

    Estas líneas deshabilitan los botones "Pausar" y "Detener" al cargar la página, ya que el temporizador aún no se ha iniciado.

  4. Función startTimer():

    5. function startTimer() {
  timer = setInterval(updateTimer, 1000);
  isRunning = true;
  empezarButton.disabled = true;
  timerInput.disabled = true;
  continuarPausarButton.disabled = false;
  detenerButton.disabled = false;
}
    • Esta función se llama cuando el usuario hace clic en el botón "Empezar".
    • Inicia un intervalo que llama a la función updateTimer cada segundo para actualizar el temporizador.
    • Establece isRunning en true para indicar que el temporizador está en funcionamiento.
    • Deshabilita el botón "Empezar" y el campo de entrada de tiempo.
    • Habilita los botones "Pausar" y "Detener".

  5. Función continuePauseTimer():

    6. function continuePauseTimer() {
if (!isRunning) {
  timer = setInterval(updateTimer, 1000);
  document.getElementById("continuarPausar").textContent = "Pausar";
  isRunning = true;
} else {
  clearInterval(timer);
  document.getElementById("continuarPausar").textContent = "Continuar";
  isRunning = false;
}
}
    • Esta función se llama cuando el usuario hace clic en el botón "Pausar/Continuar".
    • Si el temporizador no está en funcionamiento (isRunning es false), reanuda el temporizador al iniciar un nuevo intervalo y cambia el texto del botón a "Pausar".
    • Si el temporizador está en funcionamiento, lo pausa al borrar el intervalo y cambia el texto del botón a "Continuar".

  6. Función resetTimer():

    7. function resetTimer() {
  clearInterval(timer);
  document.getElementById("continuarPausar").textContent = "Pausar";
  document.getElementById("inputTime").value = "";
  updateDisplay();
  document.getElementById("display").textContent = "--";
  isRunning = false;
  remainingSeconds = 0;
  empezarButton.disabled = false;
  timerInput.disabled = false;
  continuarPausarButton.disabled = true;
  detenerButton.disabled = true;
}
    • Esta función se llama cuando el usuario hace clic en el botón "Detener".
    • Detiene el temporizador al borrar el intervalo.
    • Restablece varios elementos de la página, como el texto del botón "Pausar" y el campo de entrada de tiempo.
    • Restablece las variables isRunning y remainingSeconds a sus valores iniciales.
    • Habilita el botón "Empezar" y deshabilita los botones "Pausar" y "Detener".

  7. Función updateTimer():

    8. function updateTimer() {
if (remainingSeconds > 0) {
  remainingSeconds--;
} else {
  clearInterval(timer);
  document.getElementById("continuarPausar").textContent = "Iniciar";
  document.getElementById("inputTime").inpuContent = "";
  isRunning = false;
  resetTimer();
  return;
}
updateDisplay();
}
    • Esta función se llama cada segundo mientras el temporizador está en funcionamiento.
    • Reduce el valor de remainingSeconds en 1 segundo si todavía hay tiempo restante.
    • Si el tiempo llega a 0, detiene el temporizador, cambia el texto del botón "Pausar" a "Iniciar", restablece el temporizador y llama a la función resetTimer() para limpiar la página y las variables.

  8. Función updateDisplay():

    9. function updateDisplay() {
let minutes = Math.floor((remainingSeconds % 3600) / 60);
let seconds = remainingSeconds % 60;
document.getElementById("display").textContent = `${String(minutes).padStart(
  2,
  "0"
)}:${String(seconds).padStart(2, "0")}`;
}

    Esta función actualiza el elemento HTML con ID "display" para mostrar el tiempo restante en formato mm:ss. Calcula los minutos y segundos restantes a partir de remainingSeconds y actualiza el contenido del elemento.

  9. Escuchar el Evento de Entrada de Tiempo:

    10. document.getElementById("inputTime").addEventListener("input", () => {
let inputSeconds = parseInt(document.getElementById("inputTime").value);
if (!isNaN(inputSeconds) && inputSeconds >= 0) {
  remainingSeconds = inputSeconds;
  updateDisplay();
}
});
    • Esta parte del código escucha el evento "input" en el campo de entrada de tiempo.
    • Cuando el usuario ingresa un valor válido (un número no negativo), actualiza remainingSeconds y llama a updateDisplay() para mostrar el tiempo ingresado.

  10. Llamados de Funciones:

    11. empezarButton.addEventListener("click", startTimer);
continuarPausarButton.addEventListener("click", continuePauseTimer);
detenerButton.addEventListener("click", resetTimer);

    Aquí se establecen los controladores de eventos para los botones. Cuando se hace clic en un botón específico, se llama a la función correspondiente (startTimer, continuePauseTimer, resetTimer) para realizar la acción deseada.

Al final, con este código JavaScript dispondremos de un temporizador con funciones para iniciar, pausar, detener y reiniciar el temporizador, así como para actualizar el tiempo restante en la pantalla a cada segundo.

CSS: Estilizando el Temporizador

Finalmente, agreguemos estilos CSS para que nuestro temporizador se vea atractivo.

Para ello, agragaremos algunas clases con la propiedad class="..." y usaremos algunos id="..." para poder estilizar los elementos en nuestro CSS. A continuación nuestro archivo HTML actualizado con sus nuevas clases: 

<div class="timer">
  <h2 id="timer__title">Temporizador</h2>
  <div>
    <input type="number" id="inputTime" placeholder="00" />
    <span id="segundos">segundos</span>
    <button class="button" id="empezar">Empezar</button>
  </div>
  <div id="display">--</div>
  <div class="timer__buttons">
    <button class="button" id="continuarPausar">Pausar</button>
    <button class="button" id="detener">Detener</button>
  </div>
</div>

Ahora que tenemos nuestros elementos HTML con sus clases declaradas podemos empezar a estilizar nuestro temporizador usando nuestra hoja de estilos CSS. A continuación, un ejemplo del cómo podríamos modificar nuestro temporizador usando clases (.clase) y id's (#id): 

/* Estilos generales del temporizador */
.timer {
  max-width: 400px;
  padding: 20px;
  background-color: #f2f2f2;
  border-radius: 8px;
}

#timer__title {
  font-size: 24px;
}

#inputTime {
  width: 60px;
  font-size: 16px;
}

/* Estilos para los botones */
.button {
  font-size: 16px;
  background-color: #007bff;
  color: white;
}

/* Estilos para el reloj */
#display {
  font-size: 36px;
}

/* Estilos para el contenedor de botones */
.timer__buttons {
  margin-top: 16px;
}

Este fue un ejemplo del cómo aplicar estilos a este temporizador. Nosotros, aplicando alguno que otro estilo, modificando una cosa por aquí y por allá, obtuvimos el siguiente temporizador:

Temporizador

segundos
--

Este fue nuestro ejemplo de temporizador, pero recuerda, el límite es tu imaginación y puedes modificarlo cuanto gustes para que se adapte a tu estilo ;).
