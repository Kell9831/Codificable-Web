Calculadora con JavaScript

En este artículo, te invitamos a aprender los fundamentos de la creación de una calculadora desde cero utilizando las tecnologías esenciales de la web: HTML, CSS y JavaScript. A lo largo de este tutorial, te guiaremos paso a paso que te permitirá construir tu propia calculadora funcional desde cero. Descubrirás cómo concebir la idea, planificar la estructura y escribir el código necesario para lograr una calculadora completamente operativa.

Pasos para crear una calculadora usando HTML, CSS y JavaScript

  1. Código HTML

    
<section>
<div class="calculator__box">
  <ul class="calculator__display">
    <li class="calculator--numeroPrevio">123</li>
    <li class="calculator--operador">-</li>
    <li class="calculator--numeroActual">123</li>
  </ul>

    Creamos un seccion en el documento, en el cual agruparemos los elementos relacionados a la calculadora se crea un contenedor <div> con la clase CSS .calculator__box. Se crea otra lista desordenada <ul> con la clase .calculator__keys.

    Esta lista contiene los botones de la calculadora, que son elementos de lista <li>. Los botones representan números, operadores y funciones especiales.

    • .calculator__key:hover : Esta regla define el estilo que se aplica cuando el mouse se encuentra sobre cualquier tecla dentro del contenedor de teclas de la calculadora. Cambia el fondo a un tono más claro de violeta cuando se produce el evento hover.
    • .calculator--delete svg: Esta regla se aplica a un icono dentro de la tecla de eliminación. Establece la propiedad "pointer-events" en "none", lo que significa que el icono no responderá a eventos de puntero como clics.

  3. Código Javascript

    Programación Orientada a Objetos (POO)

    Ahora que tenemos la estructura HTML en su lugar, vamos a escribir el código JavaScript utilizando programación orientada a objetos. Crearemos una clase Calculator que contendrá todos los métodos y propiedades necesarios para realizar las operaciones de la calculadora.

    
class Calculator {
constructor() {
    // Propiedades de la calculadora
    this.numeroPrevio = "";
    this.operador = "";
    this.numeroActual = "";
    this.numDigit = 0;
    this.checkpoint = false;
    this.checkOperador = false;
    this.checkResult = false;
    this.checkDelete = false;
    this.indexOperation = -1;
    this.listOperation = ["÷", "×", "-", "+"];
    this.displayNumPrev = document.querySelector(".calculator--numeroPrevio");
    this.displayOperator = document.querySelector(".calculator--operador");
    this.displayNumActual = document.querySelector(".calculator--numeroActual");
    this.keys = document.querySelector(".calculator__keys");

    // Adjuntar eventos
    this.attachEventListeners();
}

// Otros métodos de la clase Calculator
}

    En el constructor de la clase Calculator, inicializamos las propiedades que representarán los valores y el estado de la calculadora. También seleccionamos elementos del DOM que representan la pantalla y los botones de la calculadora.

    4. Métodos de la Calculadora

    La clase Calculator tendrá varios métodos para realizar operaciones y manejar eventos. Aquí se incluyen algunos de los métodos clave:

    
redondearNumero(numero) {
  const numeroRendondeado = Number(numero.toFixed(2));
  return numeroRendondeado;
}

calcularResultado() {
  this.indexOperation = this.listOperation.indexOf(this.operador);
  let finalResult;

  if (this.numeroPrevio === "") this.numeroPrevio = "0";

  switch (this.indexOperation) {
    case 0:
      finalResult =
        parseFloat(this.numeroPrevio) / parseFloat(this.numeroActual);
      break;
    case 1:
      finalResult =
        parseFloat(this.numeroPrevio) * parseFloat(this.numeroActual);
      break;
    case 2:
      finalResult =
        parseFloat(this.numeroPrevio) - parseFloat(this.numeroActual);
      break;
    case 3:
      finalResult =
        parseFloat(this.numeroPrevio) + parseFloat(this.numeroActual);
      break;
  }

  finalResult = this.redondearNumero(finalResult);
  this.displayNumActual.textContent = "";
  this.numeroActual = "";

  this.displayOperator.textContent = "";
  this.numeroPrevio = finalResult;
  this.displayNumPrev.textContent = finalResult;

  this.checkResult = true;
}

actualizarDisplay() {
  this.displayNumPrev.textContent = this.numeroPrevio;
  this.displayOperator.textContent = this.operador;
  this.displayNumActual.textContent = this.numeroActual;
}
    • redondearNumero(numero): Este método se utiliza para redondear un número a dos decimales.
    • calcularResultado():: Calcula el resultado de una operación basada en el número previo, el operador y el número actual.
    • actualizarDisplay():: Actualiza la pantalla de la calculadora con los valores actuales.
    • attachEventListeners():: Adjunta manejadores de eventos a los botones de la calculadora para detectar clics y realizar las operaciones correspondientes.

    Eventos y Lógica de la Calculadora

    La lógica de la calculadora se controla principalmente a través de los eventos. Hemos utilizado un único manejador de eventos en la función attachEventListeners() para gestionar los clics en los botones de la calculadora. Esta función se encarga de dirigir los clics a los métodos adecuados para procesar la entrada y realizar cálculos.

    Uso de la Calculadora

    Finalmente, para utilizar la calculadora, simplemente creamos una instancia de la clase Calculator : 

    
const calculator = new Calculator();

    Esto inicializará la calculadora y la hará funcional en la página web.
